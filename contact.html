Admin Setup Instructions (Required)

1. Google Form: Create a form with 3 fields: Tracking ID, Message, and More Details. Update the form action URL and entry IDs in the code below.

2. Google Sheet: Open the Form's response sheet. Ensure Column A is Timestamp (automatic). Add a column at the far right (Column E) named Employee Response. The columns MUST be: A: Timestamp, B: ID, C: Message, D: Details, E: Employee Response.

3. Publish Sheet: In Google Sheets, go to File > Share > Publish to web. Select "Form Responses 1" (or your sheet name) and "Comma-separated values (.csv)". Click Publish.

4. Link Data: Copy that CSV link and paste it into the CONFIG.SHEET_CSV_URL variable at the bottom of this file's code.
